Ryan's Game Development Portfolio

I'm a game developer that specialises in C# and C++, I mainly use Unity to make games but I also have used Unreal Engine as well as OpenGL. I have a diploma in Game Development and Technology from Nanyang Polytechnic

  • Plooper Star Chef is a Overcooked inspired game with ragdoll physics where the player works as a chef with mini ploopers that help with cooking as well as cleaning.

  • Hooked on Hope is a 2.5D platformer game with fishing mechanics where the player is a fisherman who just wants to fish but can never seem to get actual fish.

  • This is a rhythm based platformer game where the player has to time any action to the beat of the game.

  • Nightfall Manor: Escape from the Shadowed Realm is a multiplayer co-op PVE experience where players have to work together to defeat enemies while completing tasks to escape.

  • Defenders of the Realm is a grid based tower defense game where the player has to earn money by defeating enemies to upgrade existing cannons or purchasing more to defend their home.

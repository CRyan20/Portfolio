Hooked On Hope

This game is a 2.5D platformer similar to fez, which is a my exchange program project where I worked together in Japan with people from Singapore, Japan and Finland.

I was tasked with core mechanics of the game such as the camera shifting system which allows for the perspective shift between rotations allowing for a unique way to navigate through the levels like a puzzle, the first levels level layout, the lion roar mechanic which allows the player to break through walls, the menus, sound system, as well as localisation for the game.
